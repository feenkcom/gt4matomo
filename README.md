# gt4matomo
Support for the Matomo API

## Installation

```st
Metacello new
	repository: 'github://feenkcom/gt4matomo:main/src';
	baseline: 'GToolkit4Matomo';
	load
```

## Load Lepiter

After installing with Metacello, you will be able to execute

```
#BaselineOfGToolkit4Matomo asClass loadLepiter
```
